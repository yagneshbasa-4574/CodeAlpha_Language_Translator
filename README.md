# 🌍 AI Language Translation Tool

A modern AI-powered Language Translation Tool built using **Python**, **Streamlit**, and **Google Translator API** as part of the **CodeAlpha Artificial Intelligence Internship**.

---

## ✨ Features

- 🌍 Translate text between 100+ languages
- 🔄 Swap source and target languages
- 📥 Download translated text
- 💾 Save translation history
- 📜 View translation history
- 🗑️ Clear translation history
- 🌙 Modern Dark UI
- ⚡ Fast and Easy to Use

---

## 🛠 Technologies Used

- Python 3
- Streamlit
- deep-translator
- Google Translator API

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_Language_Translator.git
```

Go to project folder

```bash
cd CodeAlpha_Language_Translator
```

Install requirements

```bash
pip install -r requirements.txt
```

Run the project

```bash
streamlit run app.py
```
